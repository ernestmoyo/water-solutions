# Water Solutions

Water infrastructure monitoring and management solutions for Africa.

## Projects

### [Izbezkal&#299;](./izbezkali/)

A comprehensive, role-based dashboard for monitoring all national water infrastructure projects across Tanzania. Features real-time metrics, geospatial maps, anomaly detection, multi-tenant support, and compliance reporting.

- **Backend:** FastAPI + PostgreSQL (TimescaleDB)
- **Frontend:** React 18 + Vite + Tailwind CSS
- **Auth:** JWT with hierarchical RBAC (Minister → CEO → Operator → Public)
- **Deploy:** Docker Compose, GitHub Actions CI/CD
